Raspberry Pi Server

Raspberry Pi sense hat

This site is powered by a Raspberry Pi 3 with a sense-hat and a WD 314Gb PiDrive. The root file system is on the PiDrive. The boot files are on the Micro SD card. The Raspberry Pi 3 uses a Broadcom BCM2837 SOC, with a 1.2 GHZ quad-core ARM Cortex A53 (ARMv8 Instruction Set) CPU, a Broadcom VideoCore IV @ 400 MHz GPU, 1 GB LPDDR2-900 SDRAM. 4 USB ports, a 10/100 MBPS RJ45 Ethernet port with 802.11n Wireless LAN and Bluetooth 4.0. The Pi also has 40 GPIO pins, a full HDMI port, a combined 3.5mm audio jack and composite video, camera interface (CSI), and a display interface (DSI).
